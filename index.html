vpn_key Восстановление работы Telegram при помощи Tor

Недавно большинство российских регионов осталось без доступа в Telegram на несколько часов.

Быстрая настройка computer

  1. Скачайте Tor Browser для Windows
  2. Запустите его и настройте мосты, если требуется. Подключитесь к сети Tor.
  3. Настройте прокси в Telegram: клик, или настройте вручную.

Просим заметить, что прокси будет работать, только когда запущен Tor Daemon (tor.exe), который запускается и останавливается вместе с Tor Browser.

Настройка tor.exe в качестве системного сервиса computer

  1. Скачайте Tor для Windows (не путать с браузером)
  2. Создайте torrc с этой строкой: 
    SOCKSPort 9050
  3. Запустите его с помощью команды 
    tor.exe --service install --options -f "D:/путь/к/torrc"
  4. Проверьте, работает ли Tor: 
    netstat -aon | findstr ":9050"

Настройка Orbot в качестве прокси в Telegram smartphone

  1. Скачайте Orbot из F-Droid или другого источника.
  2. Запустите приложение и настройте его.
  3. Перейдите на вкладку "Еще" и проверьте порт SOCKS.
  4. Настройте прокси в Telegram: клик, или настройте вручную.
